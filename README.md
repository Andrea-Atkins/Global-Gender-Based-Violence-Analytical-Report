# Global Gender-Based Violence Analysis — 2023

## Project Overview
An exploratory data analytics project examining the global severity of gender-based violence (GBV), the relationship between legal protections and prevalence rates, regional homicide trends, and whether international aid flows reflect measured need across 178 countries.

## Questions Answered
1. Where is GBV most severe globally?
2. Are countries with weaker legal protections worse off?
3. Which regions have improved and which have gotten worse?
4. Do countries with more GBV vulnerability receive proportionally more or less aid?
5. Which countries are most underserved?

## Tools Used
- Python (pandas) — data cleaning and analysis
- Microsoft Power BI — interactive dashboard
- Microsoft Excel — data organization

## Data Sources
- [WHO Global Health Observatory](https://www.who.int/data/gho)
- [UNODC Homicide Statistics](https://dataunodc.un.org/content/homicide-country-data)
- [World Bank Women, Business and the Law](https://wbl.worldbank.org/en/data/download-data)
